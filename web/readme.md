##Installaton

`npm install` (it will take some time)

##Develop Visualizer

```
cd src/visualizer
gulp serve
```

##Develop Static Site

```
cd src/static
gulp serve
```

##Build both static projects into ./dist directory

`gulp build`

##Deploy to streem.at (Github Pages)

`gulp deploy`

##Build + Deploy

`gulp build-deploy`
